## Install the library

```npm
npm i base-vaul

```

## Install a pre-styled version from the registry

This version is heavily inspired by [`shadcn drawer`](https://ui.shadcn.com/docs/components/drawer)

```npm
npx shadcn@latest add https://base-vaul.netlify.app/r/drawer.json
```
